# Alpine Legacy v0.5.2

Карьерная roguelike-RPG про альпинизм, процедурные горы, экспедиции и живой мировой спорт.

## Что есть в версии 0.5.2

- выбор любой вершины и трёх маршрутов;
- пошаговая подготовка: цель → команда → груз → выход;
- контекстное обучение первой экспедиции;
- пять игровых характеров гор;
- предпросмотр каждого шага восхождения;
- точные последствия темпа, лагеря, воды и ожидания;
- предупреждения перед тяжёлым отходом;
- обязательный спуск;
- итоговый разбор решений;
- живой мир, соперники, рекорды и новости;
- миграция сохранений v0.5.1 → v0.5.2.

## Запуск

```bash
npm ci
npm run dev
```

## Проверка

```bash
npm test
npm run build
```
