# Alpine Legacy v0.5.1

Карьерная roguelike-RPG про процедурный мировой альпинизм, одну жизнь, автономные экспедиции и историю спорта, которая развивается без игрока.

## Что работает

- процедурный горный регион и история мира по seed;
- выбор эпохи, стартового года и сложности;
- создание альпиниста, клуб, сезон и тренировки;
- выбор любой сгенерированной вершины;
- три процедурные линии на каждой горе;
- пошаговая подготовка: цель → команда → снаряжение → выход;
- понятные эффекты тренировок до подтверждения действия;
- обязательные и рекомендуемые комплекты снаряжения;
- объяснённая готовность экспедиции и кликабельные блокировки;
- маршруты, погода, команда, снаряжение и акклиматизация;
- пошаговый подъём, лагерь, ожидание погоды и обязательный спуск;
- характер, отношения, память, приказы и последствия для NPC;
- 30+ автономных мировых альпинистов и шесть клубов;
- самостоятельные экспедиции NPC при каждом продвижении времени;
- первые восхождения, рекорды скорости, травмы, смерти и отходы;
- клубные рейтинги, новости, соперники и архив рекордов;
- сохранения `v6` с автоматической миграцией карьер `v2–v5`.

## Главная навигация

- **Штаб** — состояние героя, цель и действия между экспедициями;
- **Цель** — выбор горы и маршрута;
- **Подготовка** — команда, досье, снаряжение и финальная проверка;
- **Восхождение** — активный маршрут;
- **Мир** — обзор, новости, соперники и рекорды;
- **Архив** — журнал карьеры и экспедиций.

Подготовка и мировой раздел имеют собственные внутренние вкладки. На мобильном устройстве главная навигация превращается в нижнюю панель.

## Запуск

```bash
npm ci
npm run dev
```

Проверка:

```bash
npm test
npm run build
```

## GitHub Pages

Проект использует относительный `base: './'`. Workflow находится в `.github/workflows/deploy.yml`.
В настройках репозитория выбери **Settings → Pages → Source → GitHub Actions**.

## Сохранения

- мир: `alpine-legacy:world:v1`;
- карьера: `alpine-legacy:career:v6`.

Старые карьеры мигрируют автоматически при первой загрузке.
