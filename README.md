# Alpine Legacy v0.5.3

Карьерная roguelike-RPG про альпинизм, процедурные горы, экспедиции и живой мировой спорт.

## Что есть в версии 0.5.3

- одна эталонная вершина в каждом seed-мире;
- три глубоко проработанные линии на эталонную гору;
- решения внутри маршрута, а не только выбор темпа;
- разные последствия короткой, безопасной и рискованной линии;
- стационарные верёвки, которые меняют обратный путь;
- полевые закладки с автоматическим возвратом запасов на спуске;
- отдельный маршрут спуска вместо обратного прокликивания подъёма;
- видимое время до темноты и расстояние до ближайшего лагеря;
- полный разбор выбранных линий после экспедиции;
- экспорт сейва, seed и replay-журнала;
- выбор любой вершины, подготовка, команда, снаряжение и живой мир из предыдущих версий;
- миграция сохранений v0.5.2 → v0.5.3.
