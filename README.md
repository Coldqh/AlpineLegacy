# Alpine Legacy v0.2

Карьерная roguelike-RPG про альпинизм. Игра работает как мобильное браузерное приложение и сохраняет процедурный мир и карьеру локально.

## Patch 0.2 — First Ascent

- создание одного главного альпиниста;
- выбор имени, возраста и одного из трёх происхождений;
- шесть базовых навыков;
- процедурный региональный клуб;
- наставник и клубная доктрина;
- календарь сезона;
- недельные тренировки, деньги, форма, усталость и здоровье;
- карьерный центр;
- первое квалификационное восхождение;
- пять разных участков маршрута;
- выбор темпа на каждом участке;
- детерминированные задержки, травмы и срывы;
- вершина не завершает восхождение;
- обязательный пошаговый спуск;
- отказ от попытки;
- награды только после возвращения;
- карьерный журнал и архив;
- отдельный save schema v2 для карьеры;
- совместимость с сохранённым миром v0.1.

## Основа v0.1

- процедурный мир по seed;
- три эпохи и выбор года;
- три уровня сложности;
- процедурный регион;
- 5–7 вершин;
- SVG-профили гор;
- история региона и вершин;
- мобильный и печатный интерфейс.

## Запуск

Требуется Node.js 22.12 или новее.

```bash
npm ci
npm run dev
```

Проверка:

```bash
npm test
npm run build
```

## GitHub Pages

Workflow находится в `.github/workflows/deploy.yml` и использует Node 24. В настройках репозитория выбери:

`Settings → Pages → Build and deployment → Source → GitHub Actions`

## Архитектура сохранений

- `alpine-legacy:world:v1` — мир и горный атлас;
- `alpine-legacy:career:v2` — герой, клуб, календарь, журнал и активное восхождение.

Создание нового мира удаляет привязанную к старому миру карьеру.
