# Alpine Legacy v0.11.0

Карьерная roguelike-RPG про альпинизм, экспедиции и живой процедурный мир.

## Что есть в текущей версии

- полноценное пошаговое обучение карьере и первой экспедиции;
- пять альпинистских организаций, по три разных наставника в каждой;
- наставники и другие NPC собирают полноценные группы, выбирают разные реальные маршруты, получают результаты, усталость, травмы и известность;
- каждая школа отличается специализацией, отношением к риску, качеством подготовки и стандартом восстановления;
- в каждой школе постоянно работают минимум три наставника с разным уровнем целей;
- индивидуальные навыки каждого NPC растут от реальной практики и влияют на его роль в группе;
- долгосрочная прокачка героя через тренировки, экспедиции, техническую практику и работу с наставниками;
- после тяжёлого выхода герой и NPC обязаны восстанавливаться, а снаряжение требует обслуживания;
- десять долгосрочных карьерных целей ведут от первой вершины до собственного наследия;
- единая сохраняемая экспедиция с подъёмом, обязательным спуском, лагерями, сном, спасением и последствиями;
- один общий темп вместо набора повторяющихся микрокоманд;
- разведка открывает квадрат 9 × 9 вокруг группы;
- верёвка ставится только на соседний технический участок;
- вся группа расходует силы по собственным навыкам, грузу, высоте, холоду и времени без сна;
- лагеря создаются автоматически во время полноценного сна на подходящей площадке;
- четырёхвкладочный мобильный интерфейс: подъём, 3D-гора, экспедиция и журнал;
- офлайн-PWA после первого открытия.

## Запуск

```bash
npm install
npm run dev
```
