# Alpine Legacy v0.5.5

Карьерная roguelike-RPG про альпинизм, процедурные горы, экспедиции и живой мировой спорт.

## Что есть в версии 0.5.5

- единая фирменная иконка для интерфейса, вкладки браузера, iOS и Android/PWA;
- контекстное обучение первой экспедиции без отдельной стены текста;
- три режима информации: Explorer, Climber и Expedition;
- одинаковая физика мира на всех сложностях;
- точные, округлённые или качественные прогнозы в зависимости от режима;
- более крупный и спокойный интерфейс восхождения;
- конкретные полевые детали после каждого действия;
- технический отчёт экспедиции с seed, действиями, ресурсами и причинами исхода;
- скачивание отдельных playtest-отчётов и детерминированного balance sample;
- улучшенная читаемость на телефоне;
- формат сохранений не менялся: сейвы v0.5.4 загружаются без миграции.
