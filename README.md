# Alpine Legacy v0.6.6

Карьерная roguelike-RPG про альпинизм, экспедиции и живой процедурный мир.

## Expedition UX + Logic Rebuild

- на экране похода остаются одна текущая задача и максимум четыре уместных действия;
- сложный участок проходит через понятную последовательность подготовки и движения;
- рельеф требует разной тактики, а случайное прокликивание не обходит обязательную работу;
- высотная шкала показывает старт, текущую отметку, вершину и следующую контрольную точку;
- новичок выполняет приказы и лагерную работу, но не командует экспедицией;
- автосейв и журнал оптимизированы для длинных походов в PWA;
- генерация мира на ПК показывает ошибку и позволяет повторить запуск.
