# Alpine Legacy v0.4

Карьерная roguelike-RPG про процедурный мировой альпинизм, одну жизнь, экспедиции и людей, которые помнят решения игрока.

## Что работает

- процедурный горный регион и история мира по seed;
- выбор эпохи, стартового года и сложности;
- создание альпиниста и три происхождения;
- клуб, наставник, сезон, тренировки и развитие шести навыков;
- постоянная боковая навигация с мобильным нижним режимом;
- выбор маршрута, погодного окна, команды и снаряжения;
- вес, еда, вода, топливо, акклиматизация и готовность экспедиции;
- пошаговый подъём, лагерь, ожидание погоды и обязательный спуск;
- отдельные состояния каждого участника на маршруте;
- характер, амбиции, осторожность, дисциплина, верность, эмпатия и эго NPC;
- доверие, уважение, связь, соперничество, обида и чувство долга;
- скрытые проблемы здоровья, которые могут проявиться во время выхода;
- приказы группе, их принятие и отказ выполнять;
- разворот слабого участника и назначение помощи;
- травмы, редкая гибель NPC и долгосрочные последствия;
- вкладка «Люди» с личными целями и памятью персонажей;
- структурированные отчёты экспедиций, реакция клуба и прессы;
- сохранения `v4` с миграцией карьер `v2` и `v3`.

## Запуск

```bash
npm ci
npm run dev
```

Проверка:

```bash
npm test
npm run build
```

## GitHub Pages

Проект использует относительный `base: './'`. Workflow находится в `.github/workflows/deploy.yml`.
В настройках репозитория выбери **Settings → Pages → Source → GitHub Actions**.

## Сохранения

- мир: `alpine-legacy:world:v1`;
- карьера: `alpine-legacy:career:v4`.

Старые карьеры `v2` и `v3` мигрируют автоматически при первой загрузке.
