# Alpine Legacy v0.7.1

Карьерная roguelike-RPG про альпинизм, экспедиции и живой процедурный мир.

## 3D Mountain Grid Prototype

Новый вертикальный срез подъёма вместо свободного рисования линии:

- процедурная 3D-модель горы с вращением, наклоном и приближением;
- отдельная квадратная развёртка 21 × 21;
- каждая клетка хранит высоту, уклон, рельеф, опасность и пригодность для лагеря;
- руководитель прокладывает маршрут по соседним квадратам;
- участник получает готовую линию NPC-руководителя;
- группа автоматически проходит запланированные этапы без отдельной кнопки движения на каждый шаг;
- активная пауза, ускорение времени и изменение будущего маршрута;
- три участника, ручной порядок движения, ведущий и связка;
- лагеря, отдых и закреплённые верёвки существуют на конкретных клетках;
- скрытые трещины, лавинные склоны и камнепады обнаруживаются непосредственно на карте;
- после вершины требуется проложить и пройти спуск до стартовой клетки.

Прототип доступен из главного меню и используется вместо старого экрана активной экспедиции.
