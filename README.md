# Alpine Legacy v0.1

Первая играбельная основа карьерной roguelike-RPG про альпинизм.

## Что уже работает

- музейный адаптивный интерфейс;
- главное меню;
- создание процедурного мира;
- ввод seed;
- три эпохи и выбор стартового года;
- три уровня сложности управления;
- детерминированная генерация региона;
- 5–7 процедурных вершин;
- SVG-профили гор;
- история региона и каждой вершины;
- локальное сохранение и продолжение;
- экран вершины;
- печатный режим для региона и горы.

## Запуск

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```

Готовая сборка появится в `dist/`.

## GitHub Pages

`vite.config.ts` уже использует `base: './'`. Папку `dist` можно публиковать на GitHub Pages.
