# Alpine Legacy v0.8.5

Карьерная roguelike-RPG про альпинизм, экспедиции и живой процедурный мир.

## Integrated Expedition

Подготовка и активное восхождение теперь являются одной системой. Карта получает настоящий состав, груз, запасы, погоду, навыки и состояние людей из карьеры, а каждое действие возвращает последствия обратно в сейв.

- маршрут, сторона захода, команда и снаряжение фиксируются после выхода;
- экспедиция сохраняется посреди локального этапа и продолжается с той же клетки;
- единый чистый reducer управляет движением, разведкой, верёвками, лагерями, отдыхом и исходами;
- еда, вода, топливо, метры верёвки и лагерные комплекты расходуются физически;
- усталость, травмы, обморожения, высотные проблемы, потеря груза и срывы сохраняются;
- состояние участника может перейти в травму, обездвиживание или гибель;
- добровольный отход считается полноценным решением, аварийная ситуация открывает спасательную операцию;
- вершина не завершает экспедицию: требуется пройти сохранённый маршрут обратно;
- сложности Explorer, Climber и Expedition имеют разный риск, расход сил и тяжесть последствий;
- старые активные сейвы автоматически получают новое состояние экспедиции по схеме v16.
