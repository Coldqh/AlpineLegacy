# Alpine Legacy v0.6.4

Карьерная roguelike-RPG про альпинизм, экспедиции и живой процедурный мир.

## Expedition Hardening + Content Pipeline

- три масштаба маршрутов: малая гора, серьёзная экспедиция и гигант;
- этапы собираются из terrain-модулей с разными действиями, угрозами и подготовкой;
- критические участки требуют последовательной работы, а не одной проверки;
- поражение проходит через истощение, отход, лагерь, помощь, спасение или смерть;
- автосейв создаётся после каждого действия с резервной копией предыдущего хода;
- replay проверяет последовательность и состояние экспедиции;
- контент-пайплайн валидирует ссылки, маршрут, спуск, длительность и повторяемость этапов;
- определения гор и маршрутов можно расширять без изменения симуляционного движка.

Мир остаётся нормализованным: регионы, горы, маршруты, организации, NPC и экспедиции связаны устойчивыми ID и готовы к массовому наполнению.
