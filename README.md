# Alpine Legacy v0.3

Карьерная roguelike-RPG про альпинизм. Процедурный мир, одна жизнь, подготовка экспедиции, пошаговый подъём и обязательный спуск.

## Patch 0.3 — Expedition Core

- постоянная боковая навигация на компьютере;
- компактная нижняя навигация на телефоне;
- отдельные разделы: штаб, маршрут, команда, снаряжение, экспедиция, восхождение и журнал;
- три разных маршрута на первую вершину;
- выбор погодного окна;
- акклиматизация;
- процедурная команда из самостоятельных альпинистов;
- подбор состава экспедиции;
- маршрутные требования к снаряжению;
- вес на одного участника;
- еда, вода, топливо и верёвка;
- итоговый показатель готовности с конкретными блокерами;
- полноценный выход только после закрытия плана;
- изменяющаяся погода во время восхождения;
- ночёвки, топка снега и ожидание погодного окна;
- состояние команды, время без сна и расход запасов;
- полноценный отход по пройденному маршруту;
- миграция сохранений карьеры v2 → v3.

## Запуск

Требуется Node.js 22.12 или новее.

```bash
npm ci
npm run dev
```

Проверка:

```bash
npm test
npm run build
```

## GitHub Pages

Workflow находится в `.github/workflows/deploy.yml` и использует Node 24. В настройках репозитория выбери:

`Settings → Pages → Build and deployment → Source → GitHub Actions`

## Сохранения

- `alpine-legacy:world:v1` — процедурный мир;
- `alpine-legacy:career:v3` — герой, клуб, план экспедиции, команда, снаряжение и активное восхождение.

Сохранение карьеры v2 автоматически переносится в v3. Незавершённое старое учебное восхождение закрывается при миграции, чтобы не загружать состояние с несовместимой структурой маршрута.
